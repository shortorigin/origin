# Disclosure and Public-Safety Rules

This initiative is public by default, but it is not an uncontrolled security dump.

## Publish Publicly

Publish content here when it is:

- synthetic or fully sanitized
- reproducible without live targets or secrets
- useful for secure engineering education, testing, or architecture review
- safe to run in CI and local development environments

## Keep Private First

Do not publish directly here when the material contains:

- unpatched platform-specific weaknesses
- exploit chains against live or production-relevant Short Origin surfaces
- operational secrets, access details, or infrastructure internals that increase real risk
- disclosure-sensitive findings still under triage

Report those through the private path in [SECURITY.md](/Users/justinshort/short%20origin/SECURITY.md), then publish a sanitized follow-up after remediation and approval.
