# Tutorial Chapter Template

Use this template for new guided learning paths.

## Objective

State the concrete skill the reader will gain.

## Safety Boundary

Describe what makes the exercise public-safe and reproducible.

## Inputs

List the tracked fixtures, labs, and examples.

## Guided Steps

Walk through the commands, expected output, and reasoning in sequence.

## Abuse Path

Show the bounded failure mode or adversarial behavior being studied.

## Detection and Hardening Path

Show how the defensive design detects or constrains the problem.

## Short Origin Mapping

Connect the lesson to real repository surfaces.

## Verification

List the tests or commands that prove the chapter still works.
