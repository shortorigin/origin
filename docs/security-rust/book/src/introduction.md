# Introduction

Systems Security Engineering with Rust is the public, Rust-first security corpus for Short Origin.

It combines:

- tutorials for guided, reproducible labs
- how-to guides for concrete implementation tasks
- reference pages for interfaces, tooling, and validation
- explanations that connect trust zones, controls, and runtime behavior

The v1 deep exemplar is runtime security, anchored to the existing Wasmtime sandbox, wasmCloud component metadata, evidence capture, and trust-boundary model already present in the Short Origin platform.

Use the Diátaxis sections in this book together with the repository paths under `docs/security-rust/`, `testing/security-labs/`, and `shared/` to move between concepts, code, and validation.
