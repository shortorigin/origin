# Systems Security Engineering with Rust

`docs/security-rust/` is the public, Rust-first security knowledge platform for Short Origin.

It combines four things in one governed location:

- an mdBook under `docs/security-rust/book/`
- public-safe Rust examples under `docs/security-rust/examples/`
- research notes under `docs/security-rust/research/`
- CI and contributor guidance under `docs/security-rust/scripts/`, `docs/security-rust/CONTRIBUTING.md`, and `docs/security-rust/DISCLOSURE.md`

The v1 deep exemplar is runtime security for Wasmtime and wasmCloud-aligned workloads. All other curriculum domains are scaffolded with contribution-ready placeholders so the corpus can grow through the repository's existing GitHub-native Scrumban process.
